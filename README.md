# ciweimao

刺猬猫 Go 客户端，使用安卓客户端 API。

个人用于通过 RSS 发现新书，所以没有动力实现章节相关内容，欢迎 PR。

API 解密实现基于 [Cirno-go](https://github.com/zsakvo/Cirno-go) 的解包成果。

- [ ] 账号密码登录
- [x] 获取排行榜
- [ ] 搜索书籍
- [ ] 书籍详情
- [ ] 章节内容
